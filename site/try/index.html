Using Arduino



Set up an Arduino to send and receive data over the internet using Riot.

1.

You'll need an Arduino board and an ethernet shield. We're currently working on documentation for WiFi shields. Meanwhile, the API for working with Riot is the same for both, so no harm in trying.
2.

Before you continue, you'll need an account to access the API and obtain your device connection strings. You'll also need an account to set rules (described below), view the data sent by your device and to control that device.

Request an invite. We'll email you when your account is ready.
3.

Add the pre-generated code in your account to an Arduino sketch. We'll tell you exactly how to integrate the input and output devices on your Arduino into the Riot API.
4.
